Codecademy

Design - Colors

This article by the Codecademy team talks about how color is an important tool in designing a webpage. The article goes into how to decide what colors you should pick by looking at other websites and using online tools to choose colors. The article also discusses accessibility and readability, making the user experience easy.

This article is a very useful tool for beginning web developers/designers. It gives them starting points on how they can use colors in their webpages. It also helps with letting beginners know how to accomodate colorblind and visually impared people.

IBM

User interface design for the mobile web

This article written by James Letz goes over how mobile devices differenciate between each other and the best practices for designing mobile web UIs. The article talks about display size and input for mobile devices. Some of the best practices the article mentions are using a toolkit and designing with responsive layouts and content in mind. The article also mentions pitfalls you should try to avoid when desiging mobile UIs.

This article is quite useful in the information it gives its readers. This mainly comes in the form of the things to do and avoid when desiging mobile web UIs. Even if the information might be outdated, it can provide useful as a refresher.

GeeksforGeeks

Web Development

This article on GeeksforGeeks talks about the frontend and backend development in web development. Frontend development is the part of a website users interact with, and where developers use HTML and CSS. Backend development deal with the server the website is on and doesn't interact with users.

This article is very useful in giving a short and concise explanation on the different sides of web development. This makes it great for beginners learning how to develop websites. The article also gives lists of frameworks for both frontend and backend, giving it more usefulness.

UX Magazine

Why Web Design is Dead

This article by Sergio Nouvel goes over how web design is dying out to the popularity and increase use of UX Design for a number of reasons. Some of the reasons mentions are: templates being commodofied, the use of AI, and how mobile devices decrease the need for web designers. The article then finishes off with saying designers are still needed, thanks to the rise of UX Design.

This article is an interesting read for new and experienced web designers/developers. While there are certain points the article makes that have truth, it doesn't mean web design is dying. Saying that mobile devices are killing off web designers isn't necessarily true, as many of them do design website for mobile devices.
